# project-sudoku
This project has propouse to solve the "sudoku" game through search methods.

For compile execute "python sudoku.py <algorithm> <archive_input>"

Chosen build: "python sudoku.py dfs input.txt"

    algorithm options
        - bfs
        - dfs
        - aStar

    archive_input
        - default: input.txt

## Versions
Python: 3.9.1
